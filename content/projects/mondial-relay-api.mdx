# Mondial Relay API Wrapper

The **mondial-relay** package is a high-level TypeScript library designed to simplify interactions with the Mondial Relay SOAP API. Logistics integrations are notoriously complex due to legacy protocols; this project abstracts those difficulties into a modern, type-safe developer experience. It allows businesses to easily integrate parcel tracking, point-of-delivery searching, and shipping label generation into their Node.js applications.

## Core Features

*   **Point Relais® Search**: Advanced filtering to find the nearest delivery points based on zip codes, coordinates, or city names.
*   **Label Generation**: Streamlined methods to create shipping labels and receive PDF byte streams directly from the API.
*   **Real-time Tracking**: Fetch the current status of shipments and historical tracking events using parcel numbers.
*   **Full TypeScript Support**: Comprehensive interfaces for all SOAP requests and responses, ensuring compile-time safety and IDE autocompletion.

## Technical Challenge

The primary challenge of this project was **transforming legacy SOAP/XML communication** into a predictable, modern Promise-based API. SOAP services often return complex, deeply nested objects and use non-standard error handling within the XML body itself.

To solve this, I implemented a robust mapping layer that handles the serialization and deserialization of XML using **soap** and **strong-soap**. The library automatically handles the MD5 security signatures required by Mondial Relay for every request, which involves concatenating specific parameters in a strict order. By centralizing this security logic, developers can focus on business logic rather than cryptographic formatting.

## Implementation

Below is an example of how the library handles a request to search for nearby delivery points using TypeScript.

```tsx
import { MondialRelayClient } from 'mondial-relay';

async function findNearbyPoints(zipCode: string) {
  // Initialize the client with your credentials
  const client = new MondialRelayClient({
    merchantId: process.env.MR_ID!,
    privateKey: process.env.MR_KEY!,
  });

  try {
    // Search for the 10 closest delivery points
    const points = await client.searchPoints({
      CP: zipCode,
      Pays: 'FR',
      NombreResultats: '10',
    });

    console.log('Available Points:', points.map(p => p.Ligne1));
  } catch (error) {
    console.error('Failed to fetch points:', error);
  }
}
```
